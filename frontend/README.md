# Task Analysis Dashboard

A modern, responsive dashboard for managing and analysing tasks, built with React, TypeScript, and Tailwind CSS.

## Features

- **Task Management**: Add, toggle, and delete tasks.
- **Theme Support**: Dynamic Light/Dark mode toggling.
- **Analytics**: Real-time task completion statistics.
- **Clean Architecture**: Built using Custom Hooks and Context API for scalable state management.

## 🛠 Tech Stack

- **Framework**: React (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context API & Custom Hooks
